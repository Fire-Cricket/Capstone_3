# Capstone 3: Analysis with Power BI

## Project Overview

This project is part of the Year Up United Data Analytics Training Academy Week 12 Capstone. The goal of this project is to clean, model, analyze, and visualize EmporiUm sales data using Power BI.

EmporiUm is a virtual student bookstore that sells books and technology products through both online and in-store sales. In this project, I analyzed regional sales performance over a four-year period to identify trends, top-performing categories, state-level sales performance, and top-selling general audience books.

## Business Problem

The regional sales team needs a clear way to understand sales performance across time, product categories, states, and books. This Power BI report helps territory managers, store managers, and regional leadership make better business decisions by showing key sales trends and highlighting areas of strong or weak performance.

## Project Goals

The main goals of this project were to:

* Clean and shape the provided sales data
* Build a usable data model in Power BI
* Create visuals that explain sales performance
* Identify sales trends over the full time period
* Compare sales performance by category
* Show relative sales by state
* Identify top-selling books and authors
* Present insights in a clear and professional way

## Key Insights

The report is designed to help answer questions such as:

* How have sales changed over the last four years?
* Which product categories are performing best?
* Which states are responsible for the largest share of sales?
* Which general audience books are selling the most?
* What information can help managers make better decisions?

## Data Cleaning and Modeling

During this project, I used Power Query to clean and shape the data so it could be used effectively in Power BI. I focused on making the data model clear enough to support the required visuals and business questions.

Some of the main steps included:

* Reviewing the structure of the sales data
* Cleaning fields needed for analysis
* Preparing tables for reporting
* Connecting related data
* Making sure book titles and author names could be used in the final report
